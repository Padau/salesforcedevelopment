Intro HTML
1. Wat is het verschil tussen het internet en het world wide web?
Het WWW - World Wide Web, of kortweg web, zijn de pagina's die u ziet wanneer u zich achter een apparaat bevindt en online bent.
Maar internet is het netwerk van verbonden computers waarop het web werkt, en ook welke e-mails en bestanden er doorheen gaan.
Beschouw internet als de wegen die steden en dorpen met elkaar verbinden. Het world wide web bevat de dingen die je op de weg ziet, zoals huizen en winkels.
En de voertuigen zijn de gegevens die zich verplaatsen - sommige gaan tussen websites en andere zullen uw e-mails of bestanden over het internet verzenden, los van het web.
2. Wat is het verschil tussen HTML, CSS en JavaScript?
HTML is een Hypertext Markup Language. Het is een standaard voor het bouwen van statische webpagina's en applicaties. Het gebruikt tags voor tekstbestanden om kleur, afbeeldingen, lettertypen en effecten op webpagina's te verkrijgen. Het is constant in ontwikkeling en geeft de website de structuur.
CSS of Cascading Style Sheets is een stijlbladtaal die werd gebruikt om de presentatie van de webpagina met HTML af te handelen. Het geeft de websites de uitstraling die het mooi maakt. De nieuwe versie van CSS kan ook eenvoudige animaties maken.
JavaScript of JS is een dynamische, geïnterpreteerde programmeertaal op hoog niveau die scripting aan de clientzijde mogelijk maakt. Creëert een volledig dynamische webapplicaties en websites. Het is een zeer lichtgewicht, op prototypen gebaseerde taal en werkt gemakkelijk op de browsers.
3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.
Een tag is een HTML-code die elke structuur op een HTML-pagina definieert, inclusief de plaatsing van tekst en afbeeldingen en hyperlinks. Met andere woorden, HTML-tags zijn de verborgen trefwoorden op een webpagina die bepalen hoe uw webbrowser de inhoud moet opmaken en weergeven. HTML-tags beginnen met het teken kleiner dan (<) en eindigen met schuine streep + groter dan (/>). Deze symbolen worden ook wel "angle brackets" genoemd.
Het HTML-element is alles van de start-tag tot de eindtag:
"Elementen" en "tags" zijn termen die op grote schaal door elkaar worden gehaald. HTML-documenten bevatten tags, maar bevatten de elementen niet. De elementen worden pas gegenereerd na de parseerstap, vanuit deze tags.
Een HTML-element wordt gedefinieerd door een starttag (opening). Als het element andere inhoud bevat, eindigt het met een afsluitende tag.
Voorbeeld:
                
<!DOCTYPE html>

!--opening tag for html tab-->

<body> 

<!--opening tag for body-->
<p> opening tag for paragraph </p>      <!--closing tag for paragraph-->
</body>       
<!--closing tag for body-->

</html> <!--closing tag for html-->
4. HTML gebruikt basis tags om structuur op te bouwen. De ‹ html ›, ‹ head › en ‹ body ›. Geef voor elke element aan waarvoor het dient.
<html> Defines the root of the document. It is the top level element of an HTML document. All other elements must be descendants of this element. <head> is the element container for metadata (data about data) and is placed between the <html> tag and the <body> tag. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, links to custom favicons, author, important keywords that describe the document and other meta information. <body> represents the content of an HTML document. There can be only one <body> element in a document. The <body> element contains the entire content of a webpage. It must be the second element inside of the parent <html> element, following the <head> element. It is the more important and yet very simple.
5. Op HTML elementen kun je attributen gebruiken. Wat zijn HTML attributen ? Geef minimaal 5 attributen en maak voorbeelden waar je de attributen in toepast.
De meeste HTML-tags kunnen attributen hebben. De attributen worden gebruikt om de kenmerken van een HTML-element te definiëren en worden in de openingstag van het element geplaatst.
Alle attributen bestaan ​​uit twee delen: een naam en een waarde. Atributten naam en waarden zijn niet hoofdlettergevoelig, maar het World Wide Web Consortium (W3C) beveelt echter attributen / attribuutwaarden in kleine letters aan in hun HTML 4 aanbeveling. • De naam is de eigenschap die u wilt instellen. Het element paragraaf <p> in het voorbeeld heeft bijvoorbeeld een attribuut waarvan de naam align is, dat u kunt gebruiken om de uitlijning van de paragraaf op de pagina aan te geven. • De waarde is wat u wilt dat de waarde van het onroerend goed wordt vastgesteld en altijd tussen offertes staat. Het onderstaande voorbeeld toont drie mogelijke waarden van het attribuut uitlijnen: links, midden en rechts. 5.1 Het ID-kenmerk Het id-attribuut van een HTML-tag kan worden gebruikt om elk element binnen een HTML-pagina uniek te identificeren. Er zijn twee hoofdredenen waarom u een id-attribuut voor een element zou willen gebruiken: • Als een element een id-attribuut als unieke identificatie heeft, is het mogelijk om alleen dat element en zijn inhoud te identificeren. • Als u twee elementen met dezelfde naam binnen een webpagina (of stijlblad) heeft, kunt u het kenmerk id gebruiken om onderscheid te maken tussen elementen met dezelfde naam. 5.2 Het titelattribuut Het title-attribuut geeft een voorgestelde titel voor het element. Het gedrag van dit attribuut hangt af van het element dat het draagt, hoewel het vaak wordt weergegeven als tooltip wanneer de cursor over het element gaat of terwijl het element wordt geladen.

8. Wat betekent nesting? Codeer een voorbeeld met nesting erin.
De gemakkelijkste manier om nesten te begrijpen, is door HTML-tags te beschouwen als dozen die uw inhoud bevatten. Uw inhoud kan tekst, afbeeldingen en gerelateerde media bevatten. HTML-tags zijn de dozen rond de inhoud. Soms moet u dozen in andere dozen plaatsen. Die "binnenste" dozen zijn genest in andere. In de onderste voorbeeld de <H1> H1 en <P> elementen zijn "nested" in de <body> element. 
                
                <!DOCTYPE html>
                <html>
                <body>
                
                <h1>My First Heading</h1>
                <p>My first paragraph.</p>
                
                </body>
                </html>
